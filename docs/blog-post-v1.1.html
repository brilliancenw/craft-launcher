This Free Plugin Will Change How You Use Craft CMS Forever

We built something that's made our lives a lot easier, and we wanted to share it with you.

Here at Brilliance, we work in Craft CMS all day, every day. And like you, we found ourselves clicking through the same menus over and over. You know the ones - navigating to that entry you edit constantly, checking those plugin settings, finding that Commerce order page.

We kept thinking there had to be a faster way. So we built one.

It's called Craft Launcher, and it lets you get anywhere in your Craft site with just a couple of keystrokes. We've been using it on all our projects, and honestly, it's changed how we work. We thought you might find it useful too.

Here's How It Works

Press Cmd+K (or Ctrl+K on Windows/Linux) from anywhere in the admin panel. You'll see a search interface that gives you access to everything:

Type "home" and you'll find your homepage entry right away.

Type "email" and there's your email settings.

Type "users" and jump straight to user management.

It's pretty simple, but it eliminates so much of that daily menu navigation. Once we started using it, we realized how much time we were spending just clicking around.

(Side note: We were really honored when Craft Launcher got mentioned at Dot All 2025 in Lisbon. Thanks to the amazing Pixel & Tonic team for the shout-out!)

The Smart Part

Here's what makes it really useful: it learns your workflow.

After you use it for a bit, the launcher starts tracking what you actually navigate to (not just what you search for). So when you press Cmd+K without typing anything, your most-used items are already there waiting for you.

If you edit the homepage all the time, it'll be your first result. If you're constantly checking a certain plugin's settings, it'll surface that. It basically adapts to how you work.

For us, that meant our most common tasks went from "click, click, scroll, click" to just "press Cmd+K, hit Return." Two keystrokes instead of four clicks. It adds up over the course of a day.

One Feature We're Really Excited About

We added something that's been incredibly helpful for our team: the launcher works on the front-end too.

You can enable it in your account preferences, and then when you're browsing the live site and spot something that needs fixing, just press Cmd+K, type "edit," and you're editing that exact entry.

No need to log back into the admin panel or figure out which section it's in. Just edit it right there. Our content editors have told us this feature alone makes their day so much easier.

If You're Using Blitz

This has been a big time-saver for us. When we're working on a site with Blitz caching, the launcher shows the cache status right in the search results.

So if you need to clear the cache for a specific page, you can see its status and clear it with one click - even from the front-end. That's been really handy when clients call about something not updating. We can fix it right away without digging through settings.

Same thing with View Count - if you're using it, view statistics show up right there in your search results. One less dashboard to check.

What's Included

The core features:

  • Search across entries, categories, assets, globals, users, and Commerce data
  • Quick access to plugin settings and system configurations
  • Jump to sections, entry types, and field definitions
  • Browse mode (type *) to explore your entire site systematically

The helpful extras:

  • Learns your workflow and shows your most-used items first
  • Works on the front-end for quick edits while browsing
  • Integrates with Blitz to show cache status and clear caches
  • Shows View Count stats inline with search results
  • Fully configurable keyboard shortcuts

Keyboard shortcuts we use all the time:

  • Cmd+K or Ctrl+K - Open the launcher (close it the same way)
  • Return - Go to the first result
  • Cmd+1-9 - Jump directly to a numbered result
  • * - Enter browse mode
  • Arrow keys - Navigate through results

All the shortcuts are configurable if you prefer different keys.

Security

We use this on production sites, so security was important to us. The front-end launcher:

  • Only works when you're logged in with the right permissions
  • Validates permissions for every action
  • Includes CSRF protection

It's convenient, but secure.

Want to Try It?

We've made Craft Launcher free and available in the Craft Plugin Store.

Installation takes about 30 seconds:

composer require brilliance/craft-launcher
php craft plugin/install launcher

Then press Cmd+K and give it a try.

Requirements:

  • Craft CMS 5.0 or later
  • PHP 8.2 or later

Works great with:

  • Blitz - Cache management right in your search results
  • View Count - Content analytics displayed inline
